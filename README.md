
# Install

Use curl (for Mac OS X):
~~~~
> brew install curl
> curl -o - https://raw.githubusercontent.com/iamslash/emacs_robin/master/install-all.sh | sh
~~~~

# UnInstall

Use curl (for Mac OS X):
~~~~
> curl -o - https://raw.githubusercontent.com/iamslash/emacs_robin/master/uninstall-all.sh | sh
~~~~
